# Elevate

A simple tool to run the command passed as an argument as an administrator.

## Installation

Download [elevate.exe](https://github.com/0J3/Elevate/releases/download/1.0.0/elevate.exe) and move it into a folder in your PATH.

## Building

1. Ensure Typescript is installed
2. Run `tsc`
3. Run `yarn package`
4. The EXE should be at `bin/elevate.exe`
